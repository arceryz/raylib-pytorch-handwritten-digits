# python-handwritten-digits

![](/screenshot.png)

Detect handwritten digits with Torch in Python. Uses various shitty linear models to justify moving to convolution neural networks later on. Best accuracy with a 512 to 32 linear model is 85% on MNIST data set. An experimentation ground for detecting handwritten digits, and kind-of hello world for using pytorch.

Comes with an interactive drawing tool written in *Raylib*, and is one big notebook.
